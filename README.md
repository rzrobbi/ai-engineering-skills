# Production AI Agent Skills

A collection of production-ready AI engineering skills for AI coding agents.

Designed for:

- Lovable
- Cursor
- Windsurf
- GitHub Copilot
- Claude Code

These skills transform AI coding assistants into specialized engineering roles.


## Available Skills

| Skill | Purpose |
|-|-|
| Product Architect | SaaS planning and product decisions |
| UI/UX Designer | Modern interface design |
| Full Stack Engineer | Application development |
| Database Architect | Database design and optimization |
| Security Engineer | Secure coding practices |
| QA Engineer | Testing and validation |
| DevOps Engineer | Production deployment |


## Philosophy

AI should not only generate code.

AI should think like a complete engineering team:

Product → Design → Engineering → Security → QA → Deployment


## Usage

Import any skill into your AI coding workspace.

Example:

Lovable:

Workspace Settings
→ Skills
→ Add Skill
→ Import from GitHub
